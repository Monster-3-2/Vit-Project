# Money Management Software (Tkinter)

A simple and clean desktop money management application created using Python Tkinter.  
It allows users to track expenses, incomes, budgets, and overall financial summaries.

---

## Features

### • Add Expense  
Record date, amount, and a small note.

### • Add Income  
Record income entries with date and description.

### • Delete Last Entry  
Choose whether to remove the last expense or last income.

### • View Expenses / Incomes  
Scrollable display of all transactions.

### • Set Monthly Budget  
The budget is displayed at the top and can be changed anytime.

### • Check Expenses vs Budget  
Shows whether you are within limits or overspending.

### • View All Transactions  
Shows incomes and expenses together with summary totals.

### • Clean GUI  
All buttons arranged in grid layout, responsive scroll area, and separate dialog for data entry.
